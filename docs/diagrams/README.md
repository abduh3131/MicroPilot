# Diagrams

Each `.mmd` file is a Mermaid source file. To render them as PNG images:

1. Go to [mermaid.live](https://mermaid.live)
2. Paste the contents of each `.mmd` file
3. Download as PNG
4. Save the PNG in this folder with the matching name from the table below

The markdown files reference these PNGs by name. All seven must be rendered:

| Source file | PNG name | Used in |
|---|---|---|
| `end-to-end-flow.mmd` | `end-end.png` | `README.md` |
| `system-overview.mmd` | `sysover.png` | `README.md` |
| `web-ui-flow.mmd` | `webui.png` | `README.md` |
| `adapter-architecture.mmd` | `adapter-arch.png` | `docs/MODELS_AND_ADAPTERS.md` |
| `ipc-connections.mmd` | `ipc-con.png` | `docs/IPC.md` |
| `scooter-pipeline.mmd` | `scooter-ppl.png` | `scooter/README.md` |
| `gokart-pipeline.mmd` | `gokart-pipeline.png` | `gokart/README.md` |
